How to create first website on GitHu

Step 1: Create a GitHub Account

visithttps://github.com

Sign up for a free account by filling in your details like username,email and password.After successful signup,verify your email address.

Step 2: Create a new Repository

1.Log in to your GitHub account
2.Click the + icon (top right corner)-> Select New Repository.
3. Give your repository a name ,for example : My-First-Website.
4. Set the Repository to public.
5. Check the i box: Initialize this repository with a README or Index.
6.Click create Repository
